### X-men evidence

##### Setup
```
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan db:seed
```

Admin User
```
wolverine@xmen.com
wolverine
```
> When a submission is approved the new user can access the admin using email and a temporary password `abc123`
